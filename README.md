# Umer Karamat | Personal CV Website

## Project Description
This is a personal CV website created as part of my Web Programming assignment for FAST-NUCES.  
The website showcases my personal information, education, skills, projects, and contact information.  
It is designed to be clean, professional, and responsive on both desktop and mobile devices.

## Technologies Used
- HTML5
- CSS3
- Flexbox for layout
- Responsive Design

## Website Sections
- **Header:** Name, title, profile photo, and contact information
- **About Me:** Short professional introduction
- **Education:** Academic background including degrees and percentages/CGPA
- **Skills:** Technical skills and soft skills displayed in cards
- **Projects:** Academic projects with short descriptions
- **Contact:** Email, phone number, GitHub, LinkedIn
- **Footer:** Simple copyright footer
